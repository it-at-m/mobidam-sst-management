# Swagger UI

Die Swagger UI gibt eine Übersicht über die API und ermöglicht eine einfache Testung des Backends. So können beispielsweise über den jeweiligen Controller POST, GET und DELETE Anfragen getätigt und die Antworten ausgelesen werden.

Die API kann nach dem Starten des Backends über den Link http://localhost:39146/swagger-ui/index.html#/ erreicht werden.

Weitere Informationen zur Swagger UI sind unter https://swagger.io/tools/swagger-ui/ zu finden.
